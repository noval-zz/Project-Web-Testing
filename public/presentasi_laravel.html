Sistem Keamanan — Middleware

RoleMiddleware bertindak sebagai satpam yang mengecek setiap request sebelum masuk

🌐
Request Masuk ke URL Terproteksi
Misal: /admin/dashboard atau /mahasiswa/biodata/1
↓
🛡️
Middleware: auth — Sudah Login?
Jika belum login → redirect ke / (halaman login)
if (!Auth::check()) return redirect('/');
↓
🎭
Middleware: role:... — Role Sesuai?
Jika role tidak cocok → redirect ke dashboard yang tepat (bukan error 403)
role:super_admin,admin,teknisi → /admin/* role:mahasiswa,dosen → /mahasiswa/*
↓
✅
Lanjut ke Controller
Request diproses, data diambil dari Model, View ditampilkan

  RoleMiddleware.php — Keamanan 
public function handle($request, $next, ...$roles)
{
  if (!Auth::check())
    return redirect('/'); // belum login

  if (!in_array(Auth::user()->role, $roles))
    return redirect()->route('admin.dashboard');

  return $next($request); // ✓ lanjut
}
